# Weather ETL Project

## Overview
This Python project retrieves live weather data from the Open-Meteo API, stores it in a SQLite database, and performs SQL queries to analyze the data.

## Technologies
- Python
- Requests
- SQLite
- SQL
- Git
- GitHub

## Features
- Retrieves live weather data
- Parses JSON responses
- Stores data in SQLite
- Counts weather records
- Calculates average temperature
